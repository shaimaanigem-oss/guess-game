# משחק הניחושים – איפה התמונה?

## תיאור כללי
פרויקט זה מממש משחק ניחושים אינטראקטיבי בדפדפן, שבו על המשתמש לנחש את מיקום התמונה המוסתרת בתוך לוח של 9 ריבועים (3×3).
המשחק פותח כחלק מתרגול עבודה עם טכנולוגיות Web בסיסיות: HTML, CSS ו-JavaScript, תוך דגש על לוגיקה, עיצוב וחוויית משתמש.

---

## מטרות הפרויקט
- תרגול עבודה עם מבנה דף HTML.
- שימוש ב-CSS לצורך עיצוב רספונסיבי ואפקטים ויזואליים.
- מימוש לוגיקת משחק באמצעות JavaScript.
- שילוב טיימר, ספירת ניסיונות, ומשוב למשתמש.
- עבודה עם Git ו-GitHub לצורך ניהול גרסאות והגשה.

---

## חוקי המשחק
1. הלוח מכיל 9 ריבועים (3×3).
2. בתחילת כל משחק התמונה מוסתרת באחד הריבועים באופן אקראי.
3. המשתמש לוחץ על ריבוע כדי לנסות לנחש את מיקום התמונה.
4. אם הניחוש נכון – התמונה נחשפת ומוצגת הודעת ניצחון עם מספר הניסיונות.
5. אם הניחוש שגוי – הריבוע מסומן כ״נבדק״ ולא ניתן לבחור בו שוב.
6. למשחק יש מגבלת זמן של 30 שניות:
   - אם הזמן נגמר לפני הצלחה – מוצגת הודעת `You lose` והמשחק מתאפס.
7. ניתן להתחיל משחק חדש בכל שלב באמצעות כפתור **״משחק חדש״**.

---

## טכנולוגיות בשימוש
- **HTML** – מבנה הדף והאלמנטים.
- **CSS** – עיצוב הלוח, הריבועים, אפקטי hover ואנימציות.
- **JavaScript** – לוגיקת המשחק, ניהול ניסיונות, טיימר ואירועים.
