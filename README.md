# משחק הניחושים – איפה התמונה?

משחק פשוט בו המשתמש צריך לנחש באיזה ריבוע מתוך 9 ריבועים מסתתרת התמונה.  
המשחק נבנה ב־HTML, CSS ו־JavaScript.

---

##  איך המשחק עובד?

- בתחילת כל משחק התמונה מוסתרת באחד הריבועים באופן אקראי.  
- המשתמש לוחץ על ריבועים כדי לנחש.  
- כל לחיצה נחשבת כניסיון.  
- אם המשתמש צדק – התמונה נחשפת ומוצג מספר הניחושים.  
- אם לא מתבצעת פעולה במשך **30 שניות** → מופיע “You lose” והמשחק מתחיל מחדש.  
- ניתן להתחיל משחק חדש בכל רגע עם כפתור **"משחק חדש"**.

---

##  טכנולוגיות בשימוש

- HTML  
- CSS  
- JavaScript  

---

##  קבצי הפרויקט

```
index.html
style.css
script.js
```

---

##  הפעלה

1. מורידים את כל הקבצים לתיקייה אחת.  
2. פותחים את `index.html` בדפדפן.  
3. המשחק נטען וניתן להתחיל לשחק.

---

##  הערות

- מיקום התמונה נבחר באופן אקראי בכל משחק.  
- העיצוב מותאם גם למובייל.  
- המשחק כולל טיימר של 30 שניות לכל מהלך.
